# ROADMAP — будущие планы

Файл сфокусирован только на том, что планируется добавить. Всё, что уже реализовано, описано в README.

## Тикер «Wall Street»
- Чекбоксы видимости для монет авто‑режима (по `ids`).
- Ползунок скорости + подсказка о влиянии на CPU/GPU.
- Пауза по касанию на мобильных и планшетах.
- Мягкая замена текста без скачка ширины элемента.
- Кнопка «Сбросить» для быстрых пресетов монет.

## Приложение бюджета
- Улучшения UI: адаптив, контраст, крупные касания, доступность.
- Расширенные поля: теги, вложения (фото чека), гео‑метка.
- Дополнительные графики: баланс во времени, прогнозы, сравнительные периоды.
- Экспорт: Excel, Google Sheets, расширенный CSV с метаданными.
- Резервные копии: автоматические снапшоты в файл/облако по расписанию.
- Мультивалютность: курсы, конвертация, базовая валюта в настройках.
- Автоматизация: правила категоризации по ключевым словам/источникам.
- Локализация: i18n на русском и английском с переключателем.

## Сервер и синхронизация
- JWT с обновлением токена (refresh), ограничение сессий.
- Роли в бюджете: владелец, редактор, наблюдатель.
- События SSE: расширить типы и добавить бэкофф‑логику переподключения.
- WebSocket‑режим как опция вместо SSE.
- Хранилище: переход на SQLite/PostgreSQL, миграции.

## Производительность и качество
- Профилирование фронтенда: снизить лишние перерисовки, мемоизация списков.
- Юнит‑тесты для ключевых модулей (`db`, `sync`, графики, тикер).
- Линтинг/форматирование: ESLint + Prettier (как dev‑инфраструктура).

## Инструменты разработчика
- Скрипты для сборки/миграций, единый dev‑сервер с прокси API.
- Автоматическая генерация дампов для тестовых данных.
